## intelliJ에서 git reset 하기

intelliJ의 git 창에서 `log` 탭으로 들어간다.

되돌리고 싶은 시점 오른쪽 마우스로 클릭 후 `Reset Current Branch To Here...` 선택

`git push` 하기 전에 `commit`한 걸 취소하는 건 되는데, push 하고 나서도 될 지는 모르겠다..
