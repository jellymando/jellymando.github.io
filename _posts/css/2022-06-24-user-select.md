## user-select

`user-select` 속성은 사용자가 텍스트를 선택할 수 있는지 지정한다.

값을 `none`으로 하면 텍스트를 드래그해도 텍스트 영역이 선택되지 않는다.

```css
p {
  user-select: none;
}
```
